# Cat Wiki

Cat Wiki is a web application designed to browse cat breeds and discover interesting information about them.

## Technologies Used

- React
- TypeScript
- Sass
- PHP

## Features

- Browse cat breeds
- View details and information about each breed
- Discover the top ten cat breeds
